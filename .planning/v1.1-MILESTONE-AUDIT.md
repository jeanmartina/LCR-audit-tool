# Milestone v1.1 Audit

## Result

Status: `gaps_found`

The milestone ships substantial product surface, but two end-to-end areas are still not actually complete as shipped: social/OIDC authentication is scaffolded rather than integrated, and packaged `.zip` certificate import is broken because the runtime image does not include `unzip`.

## Cross-Phase Integration

### Verified

- `invite/local-auth -> group membership -> reporting authorization` is wired.
- `certificate onboarding -> shared runtime projection -> certificate-scoped reporting` is wired.
- `reporting -> structured tags/predictive state -> localized CSV/PDF export surface` is wired.
- `compose -> web/worker/postgres/caddy` topology is wired for local/staging startup.

### Gaps

- `auth provider foundation -> real OAuth/OIDC callback delivery` is not wired end to end; the callback route remains a placeholder acceptance endpoint rather than a real provider integration.
- `batch certificate import -> packaged runtime` is not wired end to end because the runtime image does not include the `unzip` binary used by the import path.
