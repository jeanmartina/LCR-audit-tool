# Phase 21 Plan 21-01: Executive Read Models, Authorization, and Export Foundation

**Created:** 2026-04-23
**Status:** Ready for execution

## Objective

Create a richer executive reporting model with preserved group-scoped authorization and upgrade executive PDF export to use that model.

## Scope

### In Scope

- Add an executive summary read model separate from the operational grid row model.
- Preserve access for any authorized user within their allowed groups.
- Produce executive summary sections for cards, top risks, upcoming risks, trend, and simple breakdowns.
- Upgrade executive PDF export to render the richer summary.
- Extend reporting validators for the new read model/export contract.

### Out of Scope

- Final executive UI layout and print-specific CSS; Plan 21-02 owns the page surface.
- Packaged compose smoke path; Plan 21-03 owns it.
- Full SLO/burn-rate analytics.

## Threat Model

- Executive summaries must not leak data outside the principal's allowed groups.
- The executive read model must stay materially simpler than the operational grid and avoid evidence overexposure by default.
- The upgraded PDF must stay scoped to the same principal/filter context as the page.
